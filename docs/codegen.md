# Code Generation

Given a documentation build artifact, we can wholly or partially generate code out of it.
Each project uses divide and conquer to split up tasks into small enough parts to be performed
with higher reliability.

Although code generation includes code testing, additional tests in the
[Test Generation](./testgen.md#test-generation) complement code generation to ensure generated
code stays coupled and intact with the documentation.

The [Project Management](./project.md#project-management) also complements code generation well
to track the progress of the project.

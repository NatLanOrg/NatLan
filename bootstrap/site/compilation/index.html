Jazyk compiler and linker

Phases of compilation and linking powered by LLM

Compiler

Each file compiles on its own, in parallel, producing one object artifact. Deterministic steps frame the document; LLM steps read the prose.

  1. A1 Parse sections deterministic

    Markdown to a section tree. Sections hold verbatim text so the document can be reconstructed.

  2. A2 Extract entities LLM

    Identify the domain concepts a file names. Each entity gets a stable id, a scope, and an internal or external mark.

  3. A3 Extract requirements LLM

    Read each statement as an EARS requirement attached to its entities, and capture the typed edges the statement implies.

  4. A4 Consolidate relationships deterministic

    Reify the edges into relationships, deduplicate, and promote weak references to stronger UML types where the requirements warrant it.

  5. A5 Local definitions LLM

    Assemble each entity's definition from the requirements within the file. The result is one object artifact, the translation unit.

Linker

The linker takes every object artifact and resolves the program as a whole. Resolution is conservative and deterministic; semantic judgment is deferred to the validate steps.

  1. L1 Load link interfaces deterministic

    Read the entities and relationships each object artifact exports.

  2. L2 Resolve entities deterministic

    Match entities across files by explicit link, then by exact name. No fuzzy matching. An unresolved reference is a missing-link.

  3. L3 Merge relationships deterministic

    Combine the edges from every file onto the resolved entities. Produces the linked artifact.

  4. L4 Synthesize definitions LLM

    Write one coherent definition per entity from all merged requirements. Synthesizing it is itself the test of whether a merge was valid; a clash splits into a false-merge.

  5. L5 Semantic review LLM

    Read across documents for contradictions, ambiguity, and incompleteness, and record them as diagnostics with reasoning.

  6. L6 Spec-lint, coverage, reachability deterministic

    Run the structural checks over the linked graph. Produces the reviewed artifact and the combined diagnostics.
